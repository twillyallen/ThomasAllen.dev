About the NFL Prediction Simulator

An NFL Prediction tool to simulate final scores using an algorithm based on live statistical data scraped from NFL Stat websites.
Built with Python, the simulator uses NFL Team Metrics to project final scores.

Core Features:

  • Automatic Stat Collection
  • Dynamic score prediction based on a weighted algorithm
  • Projection of passing and rushing yards for both teams
  • Team selection dropdowns with official team logos
  • Option to display or hide Stat projections
  • Can save a screenshot of the prediction

Technologies Used:

  • Python
  • PyCharm
  • Tkinter (GUI Design)
  • Pandas (Data Management)
  • Custom Score Algorithms (for Game Predictions)
