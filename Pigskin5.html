About Pigskin5

This project is a daily auto-updated Trivia game in which 5 Scheduled NFL Trivia are asked.
Players have 15 seconds to answer each question, and their final score is reflected at the end.

Core Features:

  • Auto-Updated Daily Questions (JavaScript)
  • HTML and CSS Based User Interface
  • Score tracking
  • Daily Streak counter using LocalStorage
  • Share feature with custom made Copy/Paste message

Technologies Used:

  • Visual Studio Code
  • GitHub Pages
  • HTML / CSS
  • JavaScript
