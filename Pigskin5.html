Pigskin5

A Daily NFL Trivia Web App

What is Pigskin5?

Players have 15 seconds to answer each of 5 questions.
Score a 5/5 to score a Touchdown!

About

Made originally to learn JavaScript, it took off into a passion project with dozens of Daily Players.
As a huge NFL Fan, I spend a lot of time thinking of out of the box questions that only true fans
would kow the answers to.

Features:

  • Auto-Updated Daily Questions
  • Custom User Interface
  • Daily Streak Counter
  • Share feature with custom tag
  • Custom Domain

Technologies Used:

  • Visual Studio Code
  • GitHub Pages
  • HTML / CSS
  • JavaScript
