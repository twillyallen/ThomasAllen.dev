Pigskin5

A Daily NFL Trivia Web App

What is Pigskin5?

Pigskin5 is a full-stack daily NFL trivia web application that delivers a five-question timed challenge every day. Originally built as a small JavaScript learning project, it has grown into a fully-featured Web Application with hundreds of weekly play sessions, real-time leaderboard submissions, analytics tracking, and automated back-end systems powering all daily content.

Features:

  • Auto-Updated Daily Questions
  • Real-Time Leaderboard Submission
  • Custom Responsive UI for mobile and desktop
  • Daily Play Streak Counter
  • Share feature with Custom Tags
  • Custom Domain

Technologies Used:

  • Visual Studio Code
  • GitHub Pages
  • HTML / CSS
  • JavaScript
  • Google Apps Script
  • Google Sheets API
  • Google Analytics
  • JSON

What I Built

I designed and implemented:
  • A full-stack trivia system using Google Sheets as a lightweight data backend.
  • A custom REST-like API in Apps Script enabling secure score submission & retrieval.
  • A clean and scalable architecture for loading daily questions and validating answers.
  • Event-based analytics so I can track unique users, retry attempts, scroll activity, and gameplay patterns.
